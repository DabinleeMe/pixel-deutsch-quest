# 🎮 Pixel Deutsch Quest

A retro pixel-art game for learning German vocabulary and basic grammar — designed in Claude and built with AI prompting.

> ## 🚧 Work in Progress
> This project is still under active development. The core game works, but the **dialogue / conversation flow still feels a bit awkward** and needs more polishing. Expect rough edges, and check back for updates!

## ▶️ Play

**[Play it here](https://dabinleeme.github.io/pixel-deutsch-quest/)**

Or download `index.html` and open it in any modern browser.

## ✨ What's working

- Retro pixel-art visual style
- German vocabulary practice
- Core gameplay loop

## 🛠️ What still needs work

- [ ] Dialogue / conversation text feels unnatural — needs rewriting
- [ ] Smoother pacing and feedback
- [ ] More vocabulary and levels
- [ ] General polish and bug-fixing

## 📦 Tech

A single self-contained HTML file — no libraries, no build step, no external assets. Built with AI prompting.
